Qiyu

A Unity game about the survival crisis of penguins.

Camera Zoom

Zooms the Third-Person-Cam according to their sprinting status, or responds to their mouse scroll input. Utilizes Dotween for a hyper-smooth zoom animation.

Typewriter Dialog

1. Take users' input lines as a string array. 2. Forward the array index by a internal counting time, or by users' mouse click. 3. Split input string into chars. 4. Utilizes IEnumerator to print each char with a minor interval.
